# Project Title

CLI ToDo List

## Description

A simple command-line todo list manager

## Getting Started

### Dependencies

* Golang

### Installing

```
git clone https://github.com/MuhammetCanNamli/todo-list.git
```

### Executing program

* The project can be run via terminal/cmd.
  ```
  go build todo_list.go
  main
  ```
  or
  ```
  go run todo_list.go
  ```
